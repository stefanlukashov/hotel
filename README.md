Прогноз отмены бронирования отеля

Цель проекта
Предсказать, отменит ли гость бронирование отеля (is_canceled = 1) или нет (is_canceled = 0).

Датасет
- Тип: реальные данные о бронированиях двух отелей
- Размер: ~119 000 строк, 32 признака
- Целевой признак: is_canceled

Используемые технологии
- Python 3
- pandas
- numpy
- matplotlib
- scikit-learn

Модели
- Logistic Regression
- Ridge
- Random Forest
- Decision Tree

Метрики
- Accuracy
- F1
- ROC-AUC
- Precision
- Recall



Структура кода
1. Анализ датасета — статистики, гистограммы, boxplot, выводы
2. Предобработка — очистка, кодирование, нормализация, разделение выборок
3. Машинное обучение — обучение моделей, сравнение метрик, выводы
